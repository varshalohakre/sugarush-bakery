This project is a simple, responsive, and interactive front-end web application for a bakery named Sugarush. It is designed to be a lightweight alternative to complex e-commerce platforms, allowing small businesses to showcase their products effectively.

## âœ¨ Features

- **Responsive Design:** The layout adapts to various screen sizes, from desktops to mobile devices.
- **Product Gallery:** Browse through a clean, visually appealing grid of products with prices and descriptions.
- **Interactive Modals:** Click on any product to see a larger image and detailed information in a pop-up window.
- **Custom Order Form:** A dedicated page allows customers to customize their own cakes, specifying type, size, frosting, and a personal message.
- **My Orders Page:** Track and view a summary of placed orders, including their status and a total value.
- **Local Storage:** Orders are saved to the browser's local storage, so they persist even if the page is reloaded.

To run this project on your local machine, follow these simple steps:
